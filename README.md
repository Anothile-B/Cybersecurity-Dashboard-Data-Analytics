Cybersecurity Dashboard (Data Analytics Project)

*Overview*
This project is an interactive Cybersecurity Data Analytics Dashboard built using Python.
It analyzes simulated network session data to identify potential security risks, user behavior patterns, and system vulnerabilities.

The dashboard provides real-time insights into:
- Network activity
- Security threats
- User sessions
- Risk indicators

*Live Demo Link* 
View Live Dashboard : https://cybersecurity-dashboard-data-analytics-2.onrender.com/  

*Purpose*
This project demonstrates:
- Data analysis and visualization skills
- Understanding of cybersecurity concepts
- Ability to build and deploy real-world applications
- End-to-end project development (data → insights → deployment)


 *Technologies Used*
 - Python
 - Dash (Plotly)
 - Pandas
 - Plotly Express
 - Excel (Dataset)
 - Render (Deployment)

 
 *Features*
Key Performance Indicators (KPIs)
- Total Sessions
- Average Session Duration
- High Severity Sessions
- Sessions Under Attack

Visualizations
- Severity Levels of Sessions (Bar Chart)
- Protocol Usage (Bar Chart)
- Security Status Distribution (Pie Chart)
- Browser Risk Overview (Bar Chart)

Data Insights
The dashboard helps identify:
- High-risk sessions based on severity
- Suspicious login behavior
- Unusual access patterns
- Protocol usage trends
- Potential attack indicators

*How to Run* 
1. Clone the repository:
git clone https://github.com/Anothile-B/Cybersecurity-Dashboard-Data-Analytics.git

2. Navigate into the project folder:
cd Cybersecurity-Dashboard-Data-Analytics

3. Install dependencies:
pip install -r requirements.txt

4. Run the app:
python dashboard.py

5. Open in browser:
http://127.0.0.1:8050


*Deployment*
This application is deployed using Render, allowing it to be accessed online without local setup.
